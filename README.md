# PokéFinder React Application

## Project Description

PokéFinder is an interactive React web application that retrieves Pokémon data from the PokéAPI.

Users can search for a Pokémon and view information including:

- Pokémon image
- Name
- Type
- Height
- Weight


## Technologies Used

- React
- JavaScript
- CSS
- PokéAPI


## API Used

https://pokeapi.co/
